Here, we will extend the definition of \(b^x\) for exponents \(x\) that are irrational. For example, what does \(3^{\sqrt{2}}\) mean? We can approximate \(3^{\sqrt{2}}\) by using rational approximations of the irrational exponent \(\sqrt{2} = 1.41421356 \dots\text{.}\) As the approximations \(1.4, 1.41, 1.414, 1.4142\) (and so on) include more decimals, they will better approximate \(\sqrt{2}\text{,}\) and so we can expect the powers \(3^{1.4}, 3^{1.41}, 3^{1.414}, 3^{1.4142}\) to better approximate \(3^{\sqrt{2}}\text{.}\) Each of these exponents are rational and so can be calculated, and they will “converge” to \(3^{\sqrt{2}}\text{.}\) Roughly, we define \(b^r\) for irrational powers \(r\) as the limit as rational values \(x\) approach \(r\text{.}\) That is,
