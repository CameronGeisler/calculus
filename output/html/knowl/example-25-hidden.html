For \(f(x) = \sin{x}\text{,}\) \(f'(x) = \cos{x}\text{,}\) and so \(f'(0) = \cos{0} = 1\text{.}\) Then, the tangent line approximation is,
\begin{align*} \sin{x} \amp \approx \underbrace{\sin{0}}_{=0} + \underbrace{\cos{0}}_{=1} x\\ \amp = x \end{align*}
Thus,
\begin{equation*} \sin{x} \approx x \qquad \text{for } x \approx 0 \end{equation*}
