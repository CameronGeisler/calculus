For example, the derivative of \(f(x) = (u(x))^3\text{,}\) using the chain rule, is \(f'(x) = 3(u(x))^2 u'(x)\text{.}\) Using the limit definition of derivative,

\begin{align*} f'(x) \amp = \lim_{h \to 0} \frac{(u(x + h))^3 - (u(x))^3}{h}\\ \amp = \lim_{h \to 0} \frac{(u(x + h) - u(x)) \brac{(u(x + h))^2 + u(x + h) u(x) + (u(x))^2}}{h} \end{align*}

Using the limit product rule,

\begin{align*} \amp = \lim_{h \to 0} \brac{(u(x + h))^2 + u(x + h) u(x) + (u(x))^2} \cdot \lim_{h \to 0} \frac{u(x + h) - u(x)}{h}\\ \amp = 3(u(x))^2 \cdot u'(x) \end{align*}

as desired.
