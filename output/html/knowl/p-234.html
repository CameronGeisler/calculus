Find \(\frac{dy}{dx}\) if \(y^2 = x\text{.}\) This is a horizontal parabola. It defines two functions of \(x\text{,}\) \(y_1(x) = \sqrt{x}\) and \(y_2(x) = -\sqrt{x}\text{.}\) Then, for \(x \neq 0\text{,}\)
\begin{equation*} y_1'(x) = \frac{1}{2\sqrt{x}} \qquad \text{and} \qquad y_2'(x) = -\frac{1}{2\sqrt{x}} \end{equation*}
Instead, differentiate both sides of the equation with respect to \(x\text{,}\)
\begin{align*} \frac{d}{dx}(y^2) \amp = \frac{d}{dx}(x)\\ 2y \frac{dy}{dx} \amp = 1\\ \frac{dy}{dx} \amp = \frac{1}{2y} \amp\amp \text{solving for } \end{align*}
Notice that this covers derivatives for both the explicit solutions.
